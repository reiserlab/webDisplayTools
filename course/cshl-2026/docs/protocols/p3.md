## Current concept

Use an open-loop conditioning block where one object is paired with optogenetic
stimulation and another object is not, then switch to closed loop and ask
whether the fly's object behavior changes.

One candidate framing:

- Object A triggers optogenetic stimulation.
- Object B does not, or turns stimulation off.
- Students are not told which object should be "good" or "bad"; they observe
  and generate hypotheses from the behavior.
- If P1 flies look strong, a P1 chasing/object-tracking variant is a possible
  backup evening exercise.
