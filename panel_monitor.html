G6 Panel Serial Monitor

Read a panel’s USB log live in the browser — a quick “is it alive?” check.

← all web tools

⚠ Browser not supported

This browser can’t talk to serial devices. This monitor needs Web Serial, available only in Chrome or Edge on desktop. Please reopen this page there.

Connect

Plug in a panel that is running firmware (USB-serial mode — not BOOTSEL), then connect and pick its port in the chooser.

Tip: tap RUN (reset) on the panel after connecting to capture its boot banner (firmware version, predef status, clk_sys, any FATAL). Production firmware only emits logs and ignores input. The BCM self-test build adds a command console — try ? for help, i for the banner, or e0 to make the panel show the ERR glyph.
